# Blink App

"Blink App" é um aplicativo desenvolvido com o objetivo de ajudar os usuários a cuidar de sua saúde ocular. Oferece lembretes para pausar e descansar os olhos ao longo do dia, reduzindo fadiga ocular e promovendo boas práticas ao usar dispositivos digitais.

## Tecnologias Utilizadas
- Flutter: para desenvolvimento multiplataforma (iOS e Android).
- Dart: linguagem de programação utilizada no Flutter.

## Funcionalidades
- Configuração de lembretes personalizados.
- Notificações amigáveis para lembrar de piscar e descansar os olhos.
- Interface intuitiva e de fácil uso.

## Como Usar
1. Baixe o aplicativo na loja de aplicativos do seu dispositivo.
2. Abra o app e configure suas preferências de lembrete.
3. Receba notificações e cuide melhor dos seus olhos!
